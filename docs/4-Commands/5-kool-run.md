## kool run

Runs a custom command defined at kool.yaml file in the current working directory or in the kool folder of the user's home directory.

### Synopsis

Runs a custom command defined at kool.yaml file in the current working directory or in the kool folder of the user's home directory.

```
kool run [script] [flags]
```
