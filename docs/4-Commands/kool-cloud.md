## kool cloud

Interact with Kool Cloud and manage your deployments.

### Synopsis

The cloud subcommand encapsulates a set of APIs to interact with Kool Cloud and deploy, access and tail logs from your deployments.

### Examples

```
kool cloud deploy
```
